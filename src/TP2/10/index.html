Questão sobre Media Queries

Qual é a principal função das Media Queries no desenvolvimento web?

  • A) Melhorar a performance do site.
  • B) Facilitar a manutenção do código.
  • C) Adaptar o layout para diferentes tamanhos de tela.
  • D) Aumentar a segurança do site.
  • E) Melhorar a acessibilidade.

Resposta Correta: C) Adaptar o layout para diferentes tamanhos de tela.

Justificativa: As Media Queries permitem que os desenvolvedores apliquem estilos CSS específicos com base nas características do dispositivo, como largura da tela, altura, resolução, etc., garantindo que o layout se adapte de forma otimizada a diferentes tamanhos de tela.
